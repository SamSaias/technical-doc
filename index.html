Introduction

Welcome to the world of front-end web development. With only a few basic skills you can learn to create static websites which can be hosted and viewed online! This skillset can be used to land a job as a web developer, create your own personal website or blog, or simply as a hobby!

While "only a few basic skills" are necessary to start creating static web pages, there is a plethora of information available to help you hone your skills to a more advanced stack.

Prerequisites

One might say you don't need any pre-existing knowledge at all to begin working with HTML and CSS to create static web pages. The fact of the matter is, we still assume you know something.

Maybe you had a MySpace page when you were a pre-teen which you heavily decorated with sparkly green backgrounds full of raindrops and hearts. Maybe you grew up attached to a screen and can navigate your PC using only the command line. Maybe, like most of us, you are somewhere between I can turn the computer on and I know how to troubleshoot complex issues.

At the end of the day, using HTML and CSS to create a web-page requires little to no pre-existing knowledge other than what you might expect an individual inquiring in 2020 to know. I wouldn't expect my grandmother to pickup HTML and CSS, but seeing as she has never owned a cell phone or been on the internet, I also wouldn't expect her to inquire. If you can turn on a computer and use the internet to find a recipe for brownies, you can learn to use HTML and CSS!

HTML

HTML stands for Hyper Text Markup Language. Think of using a simple text document, like Microsoft Word. You have options for different elements of a page. Perhaps you have a header and footer, a title and some subtitles, bullet points or numbered lists.

These are ways in which you markup your page in order to make it representitive of how your page should be structured.

When working with code, you do not have the user interface you may be used to with a text editor. Instead you must use Hyper Text to syntactically markup your web-page. HTML defines the structure or architecture of a web page.

CSS

CSS stands for Cascading Style Sheet. As the name may suggest, CSS provides the style for your web page. Cascading is part of the name because of how a web browser reads the code written with CSS.

One would use CSS to change the layout or style of their webpage. You can add fun colors and arrange the page in lots of unique ways using CSS.

Bringing them together

While HTML and CSS are both powerful, they don't do much on their own. Technically, you can create a web-page with only HTML. It would be bland and not have much aesthetic structure, but it would still be text on a page.

CSS on it's own does absolutely nothing. You must have something to style! So, you:

  1. Set up your HTML structure. body, header, main, footer, etc.
  2. Add your content! Type...type..type!
  3. Organize your html with classes, id's, etc
  4. Add css styles!
  5. Have a static web page!
